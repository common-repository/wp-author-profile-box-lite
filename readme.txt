WP Author Profile Box Lite is an easy way to highlight author of your WordPress posts.

== Description ==

WP Author Profile Box Lite is a very easy and user friendly plugin to highlight author of your WordPress posts.
With easy option of setting and displaying the author profile on your WordPress posts, you will have no problem showing off your profile under your posts.
Since this plugin is easy to use and compatible with most of the themes based on underscores you will have no problem using it on your current running theme as the plugin is not only easy to use and compatible but 100% responsive.

== Installation ==

= Using The WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Search for `WP Author Profile Box Lite`
3. Click `Install Now`
4. Activate the plugin on the Plugin dashboard

= Uploading in WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Navigate to the 'Upload' area
3. Select `wp-author-profile-box-lite.zip` from your computer
4. Click 'Install Now'
5. Activate the plugin in the Plugin dashboard

= Using FTP =

1. Download `wp-author-profile-box-lite.zip`
2. Extract the `wp-author-profile-box-lite` directory to your computer
3. Upload the `wp-author-profile-box-lite` directory to the `/wp-content/plugins/` directory
4. Activate the plugin in the Plugin dashboard
